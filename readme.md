# AssociationChalonn

AssociationChalonn est site internet pour présenté les chats et nouvelles de l'association.

## Environnement de développement

### Pré-requis

    * PHP 7.4
    * Composer
    * Symfony 5
    * nodejs et npm

### Lancer l'environnement de développement
``````bash
composer install
npm install
npm run build

``````
